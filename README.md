# 🚀 Churn Analytics Pro

An AI-powered web application that predicts customer churn using Machine Learning and provides an interactive dashboard for business insights.

---

## 📌 Project Description

Customer churn is a major challenge for telecom and service-based companies.
This project uses Machine Learning models to predict whether a customer is likely to leave (churn) or stay.

The application is built using **Streamlit** and provides a clean dashboard where users can input customer details and get real-time predictions along with risk analysis.

---

## ✨ Features

* 🔐 Login system (basic authentication)
* 📊 Interactive dashboard with key metrics
* 🔮 Customer churn prediction
* 📈 Churn probability score
* 🚦 Risk level classification (Low / Medium / High)
* 📊 Visual charts for better understanding
* 🎨 Clean and modern UI (SaaS-style)

---

## 🛠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Joblib

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/churn-prediction-app.git
```

2. Navigate to the project folder:

```
cd churn-prediction-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
streamlit run app.py
```
